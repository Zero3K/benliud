This version of benliud is for windows mobile 6.0 , so if you want to build it from sources, you need have "windows mobile 6 standard sdk" installed. 

The visual studio c++ 2008 is necessary.

dependency:

1. openssl
2. gmp
3. tinyxml
4. sqlite

benliud is running on windows xp/vista/7, linux(ubuntu) and windows mobile, these sources are the lastest code for windows mobile.
